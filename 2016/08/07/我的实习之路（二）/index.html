我的实习之路（二）

发表于 2016-08-07   |   分类于 工作   |     |   阅读次数

以下开头是废话

从6月20号到今天，已经实习一个多月了。而我也成功从一名愣头全青转变成为了愣头半青——已经渐渐适应了公司的环境、租房的环境、工作的方式以及和他人交流的方式。所以我想就目前自己的一些经历做点儿总结，希望能够帮到其他人和接下来的自己。

去年9月份，我站在人生的十字路口……算了，这个蛮有意思的往事还是下次再说吧~

这将会是一系列连载篇，大致顺序是：

简历+面试 -> 租房 -> 困惑 -> 融入 -> 观察与总结 -> 自我提高
如有跳篇，不要惊慌，我比较随意😁

以上开头是废话

二、租房篇

经历了痛苦的简历和面试之后，终于“皇天不负有心人”，顺利拿到了offer。可是发现这才刚从一个坑爬出来却又马上掉进了下一个坑。

人生说得难听点就是不断掉坑和爬坑的过程。（祈求别掉进“粪坑”）

从刚开始找实习，我就做好了「自己找房子租」的心理准备（因为G20，你懂得……），但也仅仅只是心理准备而已。从6.20号开始工作，而学校规定7月19号之前没特殊情况的必须离校，有特殊情况的最多也只能待到7月30号。也就是说这中间还有很长的缓冲期让我能去找房子。

可是我在上了两天班之后，就开始迫不及待地想要马上租到房子了！原因有以下两点：

  1. 还是因为G20！G20期间据说会查暂住证，而我在网上看过一些悲惨故事——身为自己国家的公民，拥有合法的身份证，却因为没有暂住证......看得我简直不寒而栗，被吓得赶紧去办暂住证，可是派出所说「得先租到房子才能办暂住证」，科科。
  2. 因为太累了，而且累得毫无意义。每天早上6:30起床，晚上回寝室至少是8:30（如果运气好碰到堵车的话，科科）。每天至少有3个小时浪费在无意义的公交车上面，而有9个小时是必须呆在公司的，再减去吃饭洗澡睡觉什么的，我基本就没有什么时间可以用来看书学习了。（什么？公交车上学习？我虽然能在那种环境下坐着睡着，但是无法集中注意力看书……）

再加上联想到租房的种种好处（其实除了费点钱，剩下的全是好处）我就马上行动了。下面也就是我根据个人经历整理后的「租房子的正确姿势」。

1. 确定房租承受范围以及居住时长

「在自己的承受范围内找最好（最贵）的」，这是我买东西的信条。再说我本来就不打算在暑假实习的时候能够挣到多少钱，能够收支相抵就感动不已了，所以我最后的心理价位是1000-1200（不过后来公司说会有租房补贴500，我心理价位立马变成1500左右了(。・・)ノ）。而居住时长的话，因为不知道暑假过后是否会换一家公司，也不知道下学期导师会不会强行把我留在学校做毕业设计啥的，所以最好找能够短租两三个月的。

2. 确定租房的类型

开始想到底是自己一个人租个单间，还是找人一起合租一个套间甚至一个房间。单间和合租都各有好处：

因为1500以内在西湖区好像租不到什么好的单间，于是我便决定合租了……

3. 确定区域范围

因为我受不了把时间浪费在无意义的公交车上，所以决定在离公司半小时的路程内寻找房源。记得那天晚上我傻傻地开始在纸上画地图（异常丑），然后第二天突然醒悟，「我特么为什么不直接上网搜那片区域的地图，然后截图去打印」？

于是第二天我就这样去做了，然后把公司周围划分成了几块区域，再额外列出每块区域的小区名。这样做有很多好处：

  • 你可以借此快速熟悉公司周围环境，有什么吃的玩的就能了如指掌。(＾－＾)V
  • 当你在网上找房源的时候，一见到小区名就能快速知道它在哪个区域，是否在你的范围之内。
  • 也可以借此筛选出一些比较中意的小区进行实地考察。

想好自己的「资金承受范围」、「居住时长」、「租房类型」和「租房范围」就相当于完成了对自己能力和需求的认知。下面就可以开始寻找猎物了。

4. 寻找房源

这时候还得确定一件事情，是否能接受中介？关于中介的利弊有很多，如果抛开中介的人品问题的话，它的好处就在于能够根据你自身的条件和需求帮你快速锁定几个房子，坏处就在于你要因此付点钱给他，这个「点钱」一般是指你一个月的房租或者半个月的房租之类的。

所以如果手头宽裕，又能接受中介，建议直接找靠谱的中介机构（链家、我爱我家……），因为这样能省不少时间和精力。

那如果不能接受中介的话，就只能自己找了。我在网上搜了一些资料后，发现主要有以下几种途径：

5. 约看房

等你找到或者中介帮你找到几个比较中意的房源后，就可以约好时间去看房间了。看房间一般用不了多久，即使加上后面的谈判砍价，最多0.5-1小时就足以。所以如果你选好了几个房源，最好约在同一天的不同时间，这样对你来说最省时省力。

看房的时候一定要仔细，看看硬件和软件是否都能满足你的需要。硬件的话一般看下设备（空调、热水器、冰箱、洗衣机、厨房……）是否齐全且能正常工作就好了；而软件的话，就是看下小区周围的环境是否满意，如果是合租的话还要观察一下室友怎么样。（关于判定一个室友是否能让你满意，建议加下微信甚至微博，刷一刷他的历史记录，基本就能看出他是怎样一个人了，不过还得需要再聊聊天验证下。）

看房过程中，自己要记录下各个房源的优缺点。

6. 砍价

看房过程中就要问清楚每月价格、如何交付（付3压1什么的）、租房时长、后续转租约定。

我个人是不主张死命砍价的（因为我也不太会讨价还价），不过如果自认为砍价能力比较强，就可以试着砍下价，探探房东的底线（能省则省），但也不好太过火，毕竟买卖不成仁义在嘛~

如果没有碰到超级满意，心动到让你马上就愿意入住的房子，最好不要在看房过程中许下承诺。最好找个理由拖下时间但也不能拖太久，比如「我得回去和家人商量下，明天给你答复」之类的。

7. 判断

等看好房源，就可以揣着自己心里的那份详细对比表回学校了。最好找个纸记录下各个房源的优缺点，然后自己对比下，从中选出一个自己最满意的。（什么？选择困难症？不要骗自己了……）最后记得马上打电话给房东进行确认，以防好肉被别人抢走了。

8. 签合同/协议

最后就是找房东签合同了，这一步基本上没什么问题。仔细过一遍合同，如果没有漏洞，至此租房篇就算是结束了，接下来就愉快地搬迁吧~

哦！忘记说了，我是和同事合租的。(～￣▽￣)～那天恰好HR在群里发消息说，有个同事的室友搬出去了，需要再找一个室友，价格1400。我约他看了下房间，觉得价格、房间和他三者我都能接受，于是没怎么砍价我就决定搬过去了。（虽然有点小贵，但是我真的不想再耗费精力去找了）

总结：

  1. 确定自己的能力和需求：「资金承受范围」、「居住时长」、「租房类型」和「租房范围」
  2. 用适合自己的方式寻找房源：「中介」、「租房网」、「亲戚朋友」或「公司同事」
  3. 约好看房，然后确定自己最满意的房子，最后签署合同
  4. 搬迁……
