深夜日记-关于社交和新人问题

发表于 2016-07-05   |   分类于 日记   |     |   阅读次数

我究竟是不是有社交困难呢？应该不是……该有的情商我还是有的，一般的交际能力我也是有的，但是我却在自我抵触社交。比如说：

  • 作为一个实习生（愣头青），碰到男的就叫哥，碰到女的就叫姐，表面看来好像没什么错，而且这样既不会显得很谄媚，也不会落了礼貌和亲近。可是我心里真的很抵触，是因为自我清高在作祟？还是因为无法产生发自内心的尊敬？
  • 今天我们技术组和市场部聚餐的时候，我在想我到底要不要敬个酒呢？一直在纠结……最后还是放弃了，默默在角落开吃。遇到什么事也就附和地笑笑，基本没怎么讲话。

没想到连对别人称呼都能困扰我（如果都能直接叫名字该有多好……）。

所以，有以下想法，望我日后自己有一天能够带团队的时候能够谨记，而不是像鲁迅先生说的健忘的人一般，忘了自己曾经吃过的苦，还把这种苦让别人再尝受一遍。（这当然不是原话）

  1. 自己做团队建设的时候，首先解决称呼问题，这样不至于有太多的官僚气息也不会让新人困惑。
  2. 尽量解答新人各方面的问题，能让他畅所欲言（当然是先经过他自己观察思考的，而不是无脑问题）。
  3. 如果新人第一次聚会，一定不要让他尴尬得无所适从，主动介绍他，让他快速融入团队。
  4. 解决新人的培养模式问题，先探测底细，然后合理安排任务和进度，而不是不管他是谁，不管他的能力如何，都特么的从头来一遍，浪费大家的时间。
