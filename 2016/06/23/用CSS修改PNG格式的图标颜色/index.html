用CSS修改PNG格式的图标颜色

今天早上在看微信公众号【前端大全】的时候意外地看到了这篇文章PNG格式小图标的CSS任意颜色赋色技术,原文是张旭鑫写的，附上原文链接！

以下是我理解之后自己写的：

原理很简单，就是利用了CSS3的特性：(注意需要写前缀)

以上对所选元素进行了投影，向左偏移1px,向下偏移2px,模糊半径为3px，叠加颜色为纯红。也就是在原图左下方会出现一个红色模糊的图。

这里需要注意下filter: drop-shadow(..)和box-shadow：..的区别，drop-shadow是对图片的不透明区域进行投影，box-shadow顾名思义是对图片的盒子做投影。也就是说如果图片中间有一些透明区域，使用drop-shadow的时候会投影会保留透明区域，投射出来的和原图一样带透明，而使用box-shadow的时候投射出来的就是一个实体红色小块。说了这么多还是直接上代码看图吧:

那真的是too young,too simple!张鑫旭就遇到了这个坑，他发现只要原图在父级元素中不可见，那么投影也会消失！他后来提出了给原图添加border-right:20px solid transparent;也就是让原图的右边框以透明的形式出现在父级元素中！于是就会神奇的发现，成功了！

最后

什么！？有什么用？

  1. 不用傻逼地切两个仅仅是颜色不一样的icon了！
  2. 当icon状态因用户点击变化的时候，不用再麻烦地修改url了，只需要在css中添加好状态，然后用js改下class就OK了！
  3. 节约资源，性能优化！
