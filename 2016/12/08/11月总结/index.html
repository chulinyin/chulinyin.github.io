11月总结

发表于 2016-12-08   |   分类于 个人提高   |     |   阅读次数

从这学期以来，我的进步速度放缓了许多，想了一下大概有以下几个原因，1）找工作的事情让自己分了心。并且由于一直没有找到满意的工作，内心产生了焦虑甚至自我怀疑。2）毕业设计的事情确实挺烦人的。3）前端真的浩瀚如烟，由于没有人带，学习的目标不够明确，所以导致东学一点，西学一点的情况。4）为了让自己心智尽快成熟，想要从书中去寻找答案，可是目前还不得要领。5）自己确实有点儿懈怠了，做事不够专注，生活不够自律。

以上前四个点分别对应着我想要完成或者说有所收获的方面：工作、毕业设计、前端、思想。目前看来我一下子想要收获的东西有点太多了，以致于四个方面都没有做好。所以是时候做出一些改变了！找工作的事情我早已放下，决定等明年春招的时候再继续找。思想方面，只能先从一些小习惯开始，慢慢培养自己的思维能力。于是这个月就只剩两大目标了，不过这似乎还是会让我分心，既然如此，在这个月我就盯紧一个目标拼命往前冲，先把毕业设计搞定吧！（写点东西，确实能够帮助自己把思维理顺一点。^ - ^）

哦，光顾着写反思了，接下来写一下自己在11月干了哪些事情以及有哪些收获吧～

学习前端方面，一方面学习了 Node.js 官方文档中的几个模块。同时由于对官方文档不是很满意，自己尝试做了翻译和重构（已发在博客^_^）。目前能够用 Node.js 搭建本地服务器、处理请求和响应、与系统文件交互。另一方面就是前端基础的复习，复习了 CSS 的选择器、盒模型、文本、背景、定位和布局、变形和动画、如何优化等；并复习了原生 JS 的DOM、BOM、数据存储、通信。最后都画出了思维导图。还有就是开始对编程基础的学习，在网易云课堂的一门课程中上看了81个视频，大致了解了数据结构和算法，不过只是草草过了一遍，系统学习还是需要看书啊～

看书方面，看完了王小波的黄金时代、三十而立、似水流年，这三篇描写的是同一个人在三个不同时期遇到一些事情，总之是三篇很有趣的小说。还有就是边看边翻译了大约一章的《Beyond feelings: a guide to critical thinking》。

看电影方面，看了《奇异博士》和《比利·林恩的中场战士》。

还有就是，睡懒觉！:-) 我不知道为什么在暑假的时候，我能做到每天晚上12点左右睡觉，早上6点半起床。现在也是晚上12点左右睡觉，但是早上即使设了闹钟也还是会睡到8-9点，真的太不努力了。不行不行，明天开始早起！
