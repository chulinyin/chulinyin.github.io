遇见 MackBook Pro

发表于 2016-11-05   |   分类于 Mac   |     |   阅读次数

前天和昨天，一直在熟悉我心仪已久的 MacBook Pro 😄 ，今晚总算顺利地从 Windows 转到 MacOS 了，😄 开心 ~

很久很久以前，有位少年突然厌倦了自己正在使用的 Asus ，开始对她各种挑剔。最开始嫌弃的，是她那笨重又难看的外表，尤其是在看到 MBP 那轻盈又简约的身姿之后，这种感觉就愈发地明显。接着，他发现她的内在也不行了，动作越来越迟缓，和她一起合作的时候能够明显地感觉到她的力不从心。然而，反过来观察 MBP，他发现她是如此美妙、卓越。此时，他知道他已经爱上了 MBP ，爱到无法自拔。

但是，追求 MBP 是有代价的，对于此时的他来说，这种代价无疑是他支付不起的。正是这种约束，让他保持了耐心，继续和 Asus 合作着。但是，有耐心，最后很可能就会失去耐心。因为，只有当你做着不喜欢的事情，你才会劝说自己保持耐心。

慢慢地，他发现了 Asus 越来越多的缺点，比如：不支持蓝牙（尽管他已经准备好了蓝牙键盘和鼠标）、风扇声音大、充电器笨重、开机慢。然而，有趣的是，这些缺点，在 MBP 上全都消失不见了， MBP 那美丽的外表下，不仅有一个高雅的内在，她还支持蓝牙、即开即用、长达9小时的待机、静若处子的风扇，天呐，她怎么会如此的完美无暇？他甚至开始幻想和她一起生活的场景。

终于有一天，他知道他已经有足够地理由去追求她了，他知道那些所谓的代价再也不能阻碍他了，而恰好上天或许是被他强烈的意愿所感动了，赐给了他一次“天时”，居然让代价一下减轻了许多，他知道他不能再犹豫了，“天时地利人和”已经聚齐，他知道他终于能够拥有梦寐以求的 MBP 了，他颤抖着手指，在手机京东上激动地按下了付款按钮。

这天晚上他睡得异常安稳，他已经沉浸在即将到来的幸福海洋之中。

第二天，他起床去迎接她，那个美丽却又强大的、剽悍却又安静的她。他把她抱到了寝室，很有仪式感地打开了她。从此，他俩开始了蜜月旅行……
