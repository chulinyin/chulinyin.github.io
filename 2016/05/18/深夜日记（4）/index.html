深夜日记（4）

发表于 2016-05-18   |   分类于 日记   |     |   阅读次数

今天早上开始就一直在准备明天下午面试所需要的一些东西了，首先准备的就是自我介绍，一想到如何介绍自己我就想到了我在Beyond feelings a guide to critical thinking 9th中看到的’Individuality’那一节，里面讲到（请原谅我如下粗略的翻译理解：）

当你问一个你是谁的时候，那个人一般会先说自己的名字，如果你再问一遍你是谁，那个人可能就会说出自己的年龄、身高、体重等。然后再深入一点，那个人可能会说出自己的性格和爱好，对事物的一些看法，以及一些不为人知的秘密。也就是说描述一个人大概可以从他的生理特征、情商和智商三方面考虑。
（其实后面的内容‘How did you get the way you are? How to be a individuality?’才是重点内容，不过我只引用前面的这段作为我写自我介绍的依据）

然后我天真的就开始写
1. 自己的表面特征：
我叫XXX，是XXX大学自动专业大三的学生，想要应聘前端实习生岗位。
2. 自己的情商
爱好看书、跑步、听歌、设计和网页前端。我觉得自己做事认真靠谱，能对自己的选择负责：为了制作社团比赛和班级评比的PPT连续几天熬夜；能够独立自主，同时也有团队合作和管理经验：在担任社团技术部部长期间，管理技术部，并通过团队合作，为比赛做了充足的准备。
3. 自己的智商（能力）
我还有很强的自学能力，自学了平面设计基础，能够熟练运用ps、ai、ppt等软件；从去年九月份开始自学网页前端，报名了网易云课堂前端微专业，在今4月份完成大作业获得了优秀结业证书，现在已经能够独立完成一些简单网页的开发。
4. 结尾
所以我认为我有能力满足这个岗位的需求，能为公司创造价值，也希望公司能给我这个机会，谢谢~

当我开始沾沾自喜的时候，突然想到应该没那么简单吧？于是我就google了一下，然后看了一些“专业人士”的文章，发现我真的太想当然了。比较靠谱的自我介绍应该是：背景介绍-职位相关经历和能力-其他的一些能力和特点-我能为公司带来什么，于是我又开始重构。T——T
1. 背景介绍：
我叫XXX，是XXX大学自动专业大三的学生，想要应聘前端实习生岗位。
2. 职位相关经历和能力（这部分我最后决定采用了一个故事化的形式）
去年九月，我在翻墙找学习网站的时候，意外的学习了一个做网页的基础课程，从此我就走上了前端的不归路。为了能够更加系统地学习前端知识，我在寒假前报名了网易云课堂前端微专业，并在今年4月份获得了优秀结业证书，现在已经能够完成一些简单网页的开发。
3. 其他的一些能力和特点
我还有很强的自学能力，自学了平面设计基础，能够熟练运用ps、ai、ppt等软件；我还有较强的责任心和团队精神，做事靠谱。在大二的时候，成为了创行社团技术部部长，管理技术部并为社团提供技术支持，像制作名片、招新海报和比赛PPT。
4. 我能为公司带来什么
所以我认为我有能力满足这个岗位的需求，能为公司创造价值，也希望公司能给我这个机会，谢谢~（实在不知道能为公司带来什么大的改变）

光是为了准备这个自我介绍，就花了我两个小时的时间….好了不说了，我要熬下夜准备其他东西了，但愿明天一切顺利。
