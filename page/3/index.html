遇见 MackBook Pro

发表于 2016-11-05   |   分类于 Mac   |     |   阅读次数

前天和昨天，一直在熟悉我心仪已久的 MacBook Pro 😄 ，今晚总算顺利地从 Windows 转到 MacOS 了，😄 开心 ~

很久很久以前，有位少年突然厌倦了自己正在使用的 Asus ，开始对她各种挑剔。最开始嫌弃的，是她那笨重又难看的外表，尤其是在看到 MBP 那轻盈又简约的身姿之后，这种感觉就愈发地明显。接着，他发现她的内在也不行了，动作越来越迟缓，和她一起合作的时候能够明显地感觉到她的力不从心。然而，反过来观察 MBP，他发现她是如此美妙、卓越。此时，他知道他已经爱上了 MBP ，爱到无法自拔。

但是，追求 MBP 是有代价的，对于此时的他来说，这种代价无疑是他支付不起的。正是这种约束，让他保持了耐心，继续和 Asus 合作着。但是，有耐心，最后很可能就会失去耐心。因为，只有当你做着不喜欢的事情，你才会劝说自己保持耐心。

慢慢地，他发现了 Asus 越来越多的缺点，比如：不支持蓝牙（尽管他已经准备好了蓝牙键盘和鼠标）、风扇声音大、充电器笨重、开机慢。然而，有趣的是，这些缺点，在 MBP 上全都消失不见了， MBP 那美丽的外表下，不仅有一个高雅的内在，她还支持蓝牙、即开即用、长达9小时的待机、静若处子的风扇，天呐，她怎么会如此的完美无暇？他甚至开始幻想和她一起生活的场景。

终于有一天，他知道他已经有足够地理由去追求她了，他知道那些所谓的代价再也不能阻碍他了，而恰好上天或许是被他强烈的意愿所感动了，赐给了他一次“天时”，居然让代价一下减轻了许多，他知道他不能再犹豫了，“天时地利人和”已经聚齐，他知道他终于能够拥有梦寐以求的 MBP 了，他颤抖着手指，在手机京东上激动地按下了付款按钮。

这天晚上他睡得异常安稳，他已经沉浸在即将到来的幸福海洋之中。

第二天，他起床去迎接她，那个美丽却又强大的、剽悍却又安静的她。他把她抱到了寝室，很有仪式感地打开了她。从此，他俩开始了蜜月旅行……

前言

最近在看 Node.js v4.6.1 的官方文档，发现有点儿难啃。我认为是官方文档介绍顺序的问题（当然可能是我太笨了），它没有对 API 作适当的分类和排序，只是按照字母顺序粗暴地罗列了API，这导致文档条理不清晰，看起来比较费劲。因此，我对几个我认为比较重要的模块进行再整理分类，下面是一些笔记。（如果嫌啰嗦，直接看应用部分和总结部分即可。）

额，突然发现官方稳定版变成 v6.9.1 了，好在这个模块几乎没有多大变化。(＾－＾)V

概念

Client) : 客户端，是一块计算机硬件或软件，可以访问由服务器提供的服务。服务器通常在另一个计算机系统上（也可能在本地），在这种情况下，客户端通过网络的方式访问服务。是一个计算机程序，其操作的一部分依靠发送请求到另一个计算机程序（可能在也可能不在另一台计算机上）。比如，web 浏览器就是一个客户端，可用于连接 web 服务器并显示网页。

Server : 服务器，是一个计算机程序或者设备，用于为其他程序或者设备（客户端）提供功能。一个管理资源并为用户提供服务的计算机软件。

web server : 网络服务器是一个通过 HTTP 处理请求的计算机系统。可以指整个系统，或者特指接收并管理 HTTP 请求的软件。

HTTP : HyperText Transfer Protocol ，超文本传输协议。 是在万维网上进行通信时所使用的协议方案。 Web 使用其作为规范，完成客户端到服务器端等一系列运作流程。

Agent : 代理（客户端），发起自动 HTTP 请求的半智能 Web 客户端。通常指客户端程序，如浏览器就是一个 Agent 。

Proxy : 代理，位于客户端和服务器之间的 HTTP 中间实体。接收由客户端发送的请求并转发给服务器，接收服务器返回的响应并转发给客户端。

《思维的乐趣》——知识分子的不幸

发表于 2016-10-15   |   分类于 读书随感   |     |   阅读次数

开始第二遍看《思维的乐趣》了，我会对我觉得有意思的部分做摘录，然后讨论一番，希望能够帮助我在思维中享受到某些乐趣。

什么是知识分子最害怕的事？而且我有答案，自以为经得起全球知识分子的质疑，那就是：“知识分子最怕活在不理智的年代。”所谓不理智的年代，就是伽利略低头认罪，承认地球不转的年代，也是拉瓦锡上断头台的年代；是茨威格服毒自杀的年代，也是老舍跳进太平湖的年代。我认为，知识分子的长处只是会以理服人，假如不讲理，他就没有长处，只有短处，活着没意思，不如死掉。

什么叫理智？——理性智慧？摆事实，讲道理，谈利害？

但知识分子赶上这么个年代，死活不是问题。最大的问题是：这个倒霉的年头几何时过去。假如能赶上这年头过去，就活着；赶不上就犯不着再拖下去。……就我所知，他们最关心的正是赶得上赶不上的问题。在那年头死掉的知识分子，只要不是被杀，准是觉得赶不上好年头了。而活下来的准觉得自己还能赶上——当然，被改造好了、不再是知识分子的人不在此列。

我们所说的不理智，到底是因何而起？对此我有个答案，但不愿为此打赌，主要是怕对方输了赖账：此种不理智，总是起源于价值观或信仰的领域。对于这世界上的各种信仰，我并无偏见，对有坚定信仰的人我还很佩服，但我不得不指出，狂信会导致偏执和不理智。

因为狂信，人就不想讲道理。我个人以为，无理可讲比尸横遍野更糟；而且，只要到了无理可讲的地步，肯定也要尸横遍野，“文化大革命”里就死人不少，还造成了全民知识水平的大倒退。

当然信仰不是总要导致狂信，它也不总是导致不理智。全无信仰的人往往不堪信任，在我们现在的社会里，无信仰无价值的人正给社会制造麻烦，谁也不能视而不见。

我的老师说：对一般人来说，有信仰比无信仰要好。

《思维的乐趣》——中国知识分子与中古遗风

发表于 2016-10-06   |   分类于 读书随感   |     |   阅读次数

开始第二遍看《思维的乐趣》了，我会对我觉得有意思的部分做摘录，然后讨论一番，希望能够帮助我在思维中享受到某些乐趣。

我们国家总以受过某种程度的教育为尺度来界定知识分子，外国人却不是这样想的。他们认为工程师、牙医之类的人，只能算是专业人员，不能算知识分子，知识分子应该是在大学或者研究部门供职，不坐班也不挣大钱的那些人，还要“批判社会”。

维基百科中说：知识分子主要是指具有相对丰富知识，自由思想，独立人格，对于现实持有一定批判精神的人士。在中国大陆知识分子不被视为一种阶级成分。[1]

一般说来，现代关于知识分子有三种定义。第一种定义是指以从事思想、著作以及精神生活等方面职业为主的人士；第二种定义主要来自马克思主义，知识分子是指从事教学、律师、新闻等职业的人员，这种定义在中国大陆比较流行；第三种定义指文化知识分子，主要指具有文化、艺术方面特殊才能的人，因其才能获得某种话语权，从而能够以这种话语权影响公众和其它事物的人士。

维基百科中还有一句比较有意思：在中国，秦汉大一统之后，皇权无可逃避于天地间，国家强过社会，士大夫都是官吏，读书人虽多，知识分子却绝少。

中国知识分子关注社会的伦理道德，经常赤膊上阵，论说是非；而外国的知识分子则是以科学为基点，关注人类的未来，就是讨论道德问题，也是以理性为基础来讨论。人家那里热衷于伦理道德的，主要是些教士，还有一些是家庭妇女。大学教授绝不会站在讲坛上说：你们这些罪人，快忏悔吧……这与身份不符。因为口沫飞溅，对别人大作价值评判，层次很低。

我国的知识分子在讨论社会问题时，常说的一件事就是别人太无知。一般来说，知识的多寡是个客观的标准，但把自编的黑话也列入知识的范畴，就难说有多客观了。（不知道学问，死记些名字，有何必要？）还有一个爱说的话题就是别人“格调低下”，我以为这句话的意思是说：”兄弟我格调甚高，不是俗人！”（挑别人毛病要靠谱，不要借着贬别人来抬自己。）

如果把道德伦理的功能概括为批判和建设两个方面，以上所说的属于批判方面。我不认为这是批判社会——这是批判人。知识分子的批判火力对两类人最为猛烈：一类是在校学生，尤其是中学生；另一类是踩着地雷断了腿的同类。这道理很明白——别人咱也惹不起。

知识分子倾向于批判社会，而我国的“知识分子”倾向于批判人？

我们当然可以潜心于伦理学、道德哲学，营造一批道德体系，供社会挑选，或是向社会推荐——但是这件事几乎没有人干。现在的知识分子都学乖了，只管呼吁不管干，并且善用一种无主句：“要如何如何”。

中国现代知识分子的遗风是什么？他既不像远古的中国知识分子（如孔孟、杨朱、墨子）那样建立道德体系，也不像现代欧美知识分子跨价值观地立论（价值中立）。最爱干的事是拿着已有的道德体系说别人，如前所述，这正是中古的遗风。倒霉的是，在社会转型时期，已有的道德体系不完备，自己都说不清；于是就哀叹：人心不古，世道浇漓，道德武器船不坚，炮不利，造新船新炮又不敢。其实可以把开船打炮的事交给别人干——但咱们又怕失业。当然，知识分子也是社会的一份子，也该有公民热情，针砭时弊也是知识分子该干的事；不过出于公民热情去做事时，是以公民的身份，而非知识分子的身份，和大家完全平等。这个地位咱们又接收不了，非要有点知识分子特色不可。照我看这个特色就是中古特色。

现在中国知识分子在关注社会时，批判找不找目标，颂扬也找不着目标，只一件事找得着目标：呼吁速将大任降给我们，这大任乃是我们维护价值体系的责任，没有它我们就丧失了存在的意义。要论价值体系的形成，从自然地理到生活方式都有一份作用，其功能也是关系到每一个人，维护也好，变革也罢，总不能光知识分子说了算呐。要社会把这份责任全交给你，得有个理由。说来说去只能说凭我清楚明白。那么我只能凭思维能力来负这份责任，说那些说得清的事；把那些说不清的事，交付给公论。现代的欧美知识分子就是这么讨论社会问题：从人类的立场，从科学的立场，从理性的立场，把价值的立场剩给别人。咱们能不能学会？

批判为什么会找不着目标呢？是即使找着目标也要权衡利弊，不敢说出口吧？

颂扬为什么也找不着目标呢？因为真的没有什么值得颂扬的。

可是我们的“知识分子”（那些明明算不上，却自诩如是的人）喜欢先把价值的立场给抢占了？

最后说说中国知识分子的传统。当然，他有“士”的传统。有人说，他先天下之忧而忧，后天下之乐而乐（悲观主义者？），有人说，他以天下为己任（国际主义者？），我看都不典型。最典型的是他自以为道德清高（士有百行），地位崇高（四民之首），有资格教训别人（教化于民）。这就是说，我们是这样看自己的。问题是别人怎样看我们。

这就不光光是知识分子了，许多人都自以为道德清高，有资格教训别人呢！内心的道德优越感，犹如毒品一般，让他们痴狂。

如果我们讨论社会问题，就讲硬道理：有什么事，我知道，别人还不知道；或者有什么复杂的问题，我想通了，别人想不通；也就是说，按现代的标准来表现知识分子的能力。这样虽然缺少了中国特色，但也未见得不好。

不知道这种现象或者说讨论氛围什么时候才能开始在我们国家弥漫。

思维的乐趣

发表于 2016-10-04   |   分类于 读书随感   |     |   阅读次数

开始第二遍看《思维的乐趣》了，我会对我觉得有意思的部分做摘录，然后讨论一番，希望能够帮助我在思维中享受到某些乐趣。

这种痛苦（得不到思维的乐趣）的顶点不是被拘押在旅馆里没有书看、没有合格的谈话伙伴，而是被放在外面，感到天地之间同样寂寞，面对和你一样痛苦的伙伴。在我们之前，生活过无数的大智者，比方说，罗素、牛顿、莎士比亚，他们的思想和著述可以使我们免于这种痛苦，但我们和他们的思想、著述，已经被隔绝了。

痛苦在有了对比之后会表现得更加明显。因为一个人在一个密闭空间的时候，你或许还能够自我安慰，想象所有人都是这样生活的，但当你意识到仅仅是你或者包括你在内的一部分人是这样的的时候，这种强烈的对比和反差，才让人绝望。有些事情，比死亡更加让人恐惧。

一个人倘若需要从思想中得到快乐，那么他的第一个欲望就是学习。

这正是因为在那些年代，有人想把中国人的思想搞得彻底无味。我们这个国家里，只有很少的人觉得思想会有乐趣，却有很多的人感受过思想带来的恐慌，所以现在还有很多人以为，思想的味道就该是这样的。

那些人为什么想把中国人的思想搞得彻底无味？那些运动对他们来说有什么好处吗？为了能够更加方便地统治？他们需要听话的人？对呀，谁不想要一群听话的人供自己差使呢？他们最怕的或许就是那群人开始思考。

这就是所谓的“一朝被蛇咬，十年怕井绳”？记得在鲁迅先生的书中看到过类似的故事，如果你是一只猫，你就不能太敏感，否则你有一次被坑头烫伤后，你就再也享受不到坑头的温暖了。

然而，某个人被剥夺了学习、交流、建树这三种快乐，仍然不能得到我最大的同情。这种同情我为那些被剥夺了“有趣”的人保留着。

关于《恐怖直播》

发表于 2016-10-01   |   分类于 观影随感   |     |   阅读次数

前几天看了一部韩国电影——《恐怖直播》，由于剧情十分紧凑，看了一遍之后觉得有些地方还是不太懂，因此马上又看了第二遍，并摘录了一些台词。

大致剧情简介：

男主是一个十分有才华的人，却因为遭人（领导）排挤，从新闻主播变成了广播电台的主持。突然有一天，他接到了一通奇怪的连线，对方（男2）威胁说要用炸弹炸桥。男主当时十分恼火，并以为这是一个无聊的恶作剧。可是没过几分钟，公司附近的大桥真的被炸弹炸断了，所有人都炸开了锅，开始猜测爆炸的原因。男主第一时间想到打电话报警，可是在拨通电话的瞬间，他意识到这是一个独家新闻，同时也是他翻身的好机会。

于是，男主挂断电话，并打电话给领导，要求让他直播这个独家新闻，而领导为了收视率答应了他。于是剧情的高潮部分就开始了。

男主2声称自己是一名劳苦工人，拼命地为国家卖命，可是有一天他的几个工友在晚上赶工翻新大桥的时候，不小心从桥上掉了下去，由于政府的不作为，他们最后死掉了。所以男2想通过这个方式，让总统给他们道歉。

男主听后当然觉得不可能，可是谁知男主耳麦上被安了炸弹，于是在利益驱使以及生命威胁下，男主一步步走进了男2精心设好的圈套。可是男主和男2都没有想清楚的是，总统不可能在这种胁迫下，公开在电视台道歉。于是总统就给他们设了一个必死局。

关于JS中的布尔操作符

发表于 2016-09-24   |   分类于 JavaScript   |     |   阅读次数

前言

今天开始第二遍看高程了，当看到布尔操作符的时候，我就彻底被它搞晕了。书上所描述的规则过于繁琐，比如逻辑与那里：

  • 如果第一个操作数是对象，则返回第二个操作数；
  • 如果第二个操作数是对象，则只有在第一个操作数的求值结果为true的情况下才会返回该对象；
  • 如果两个操作数都是对象，则返回第二操作数；
  • 如果有一个操作数是null/NaN/undefined，则返回null/NaN/undefined；（这句话是错的！false && null的结果是false）
  • ……

然而这才只是逻辑与啊！还有逻辑或，算了，我不列了。总之，这部分描述地实在太过于啰嗦。所以有感而发，写个简明易懂版。

Boolean类型

想要了解布尔操作符，最好先了解一下JS中的基本类型——Boolean类型，该类型只有两个值：true和false。

由于JS是一门弱类型的语言，所以在做一些计算或者判断的时候，会存在隐式类型转换。比如在做if()条件判断的时候，其中的值会被转换成为布尔值。所以实际上JS中所有类型的值（或者说变量）都可以转换为布尔值。那么哪些值转换后会是true，哪些值转换后会变成false呢？

  • 会被转为false的值：undefined、null、false、0、NaN、空字符串。
  • 会被转为true的值：除了会被转为false的值(＾－＾)V，如：true、除0和NaN以外的任何数字、任何对象、任何非空字符串。

如果想把一个变量显示地转换成布尔值，可以通过Boolean(variable)或者!!variable。

逻辑非（!）

看了前言，再看逻辑非操作符就很简单了，! variable 其实就是 ! Boolean(variable)。也就是只要把转换为布尔值后的结果再反一下即可。

逻辑与（&&）、逻辑或（||）

expr1 && expr2的结果，其实一句话即可概括：

如果expr1 能转换成false（Boolean(variable)===false），则结果为expr1，否则结果为expr2。

同理expr1 || expr2的结果：

如果expr1 能转换成true（Boolean(variable)===true），则结果为expr1，否则结果为expr2。

注意：expr1和expr2不一定为布尔值，也就是说，结果可以是其他类型的值，如null && 2的结果为null、1 && 'cc'的结果为'cc'、'cc' || 2的结果为'cc'、'' || [1,2]的结果为[1,2]。

短路操作

除此以外，还需要额外注意一下短路操作：

如果第一个操作数能够决定结果，那么就不会再对第二个操作数求值。

沉默的大多数

发表于 2016-09-22   |   分类于 读书随感   |     |   阅读次数

开始第二遍看《思维的乐趣》了，我会对我觉得有意思的部分做摘录，然后讨论一番，希望能够帮助我在思维中享受到某些乐趣。

人要永远做小孩子虽办不到，但想要保持沉默是能办到的。在公众场合什么都不说，到了私下里则妙语连珠，换言之，对信得过的人什么都说，对信不过的人什么都不说。（中国人的通病？）

所以这里的保持沉默是在相对情况而不是绝对情况下的吗？对信不过的人保持沉默，对信得过的人，畅所欲言。那如何判断一个人是否信得过？

我所理解的“信得过”是指，你相信他不会把你的言论传播出去，即使传播也必定有所保留，不会把你暴露，总而言之就是不会让你受损，甚至会让你受益。然而受损和受益的判断标准又是相对的，每个人的标准有所不同。比如说，他可能认为使你头发短些是无损的，甚至是有好处的，可是你可能认为这样会让你受损。所以，信得过也是相对的？

然而按作者所说，他把人泾渭分明地分成两类，然后对之以完全相反的两种态度，真的能够做得到吗？判断信得过的依据又是什么呢？

我自问做不到如此，而且我认为作者想要说明的可能是“沉默和信得过应该成反比，也就是对越信不过的人，越保持沉默”。可是这样一来情况变得更加复杂了，因此我这个认为极有可能是错误的。所以，我现在困惑的是，分界线到底在哪里？纯粹依靠个人直觉吗？
