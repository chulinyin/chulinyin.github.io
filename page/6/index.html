关于《三人行》

今天晚上和朋友去看了一部电影《三人行》，其实本来应该是昨天晚上和他去看《惊天魔盗团2》的，但由于突然有事就不能去了，于是他就和一个妹子一起去看了。其实本来我对国内这种题材的电影已经不感兴趣了，不想去看的，但是朋友很喜欢钟汉良，于是就……

提前5分钟进场，然后就坐那里看了5分钟的广告！那么low的广告简直无法忍受！不知道算不算是侵犯消费者的权益，因为我是付费看电影去的，不是看广告去的，而且国内的广告质量真的不敢恭维。

好了，终于等到正片了！开场是赵薇扮演的医生正在给一个病人做开颅手术，画面还是有点血腥的。然后情节就开始一路让我看不懂了，先是赵薇被一个因她手术而截瘫的病人羞辱，然后突然古天乐扮演的警察和钟汉良扮演的劫匪出现了。钟汉良因为警察的走火脑袋里进了一颗子弹，而古天乐为了包庇自己的同事不惜做伪证，同时还想方设法抓住钟汉良的同伙。

  • 钟汉良的这个角色很有意思，台词很有哲学家的味道，引用了好多的哲学故事，也恰当地运用了哲学思想。
  • 赵薇的这个角色一开始是一个执拗的性格，还天真地本着医生的职责帮钟汉良给他同伙打了电话，在被古天乐扇了几个巴掌之后居然又完全颠倒想去害死钟汉良。
  • 古天乐的这个角色很有警察的风范，但他却选择包庇同事。他有一句台词让我印象深刻：犯法是为了执法。

电影情节很混乱，我还是没有缓过来。本应该是电影高潮的枪战片段，也被弄得很让人看不懂，明明是惊险刺激的枪战配乐却是一首莫名其妙的歌，然后有很多慢镜头和特效。

最后的结局更有意思，古天乐居然选择救了钟汉良，赵薇的那个截瘫病人选择滚下楼梯自杀之后居然莫名站了起来可以走路了（当然这里我极度怀疑是赵薇的幻觉，是她自我内心的一种救赎吧，就像《心理罪》里男主还以为女朋友还活着）。

最后的最后是，古天乐向上司承认做伪证，赵薇抛开了心结开启音乐做起了手术，钟汉良变成植物人躺在病床上。

总之我看得很迷糊……

window.sessionStorage的妙用（2）

发表于 2016-06-22   |   分类于 JavaScript   |     |   阅读次数

最近慢慢悟到：特性一定对应着相应的需求，如果你看到一个API或者对象有某些你不了解的特性，尝试去想下它对应的需求是什么，或者说它可以实现什么需求？

昨天说到,将window.sessionStorage用于页面误刷新之后的表单内容恢复操作，然后还悄悄卖了个关子。今天就来探索它恢复(回退)页面后，数据依然保存着的特性的妙用！

当我们顺着APP逻辑逐级向下走的时候，我们可能会需要返回上一级的菜单去重新选择另一个页面（分支）。那当我们返回上一级菜单或者页面的时候，我们可能想要让返回页面出现在我们上次浏览到的位置而不是再重头向下滑动屏幕。这样讲好像不是很生动，还是举个例子吧~比如：

我们在上网购物的时候，一般会在一个商品简介列表页面中寻找中意的商品。向下滑动屏幕滑了半天，终于发现一个比较满意的，然后点击进去看详情发现“我靠！居然是个标题党！”，于是怒戳返回按钮……
注意：这时候如果发现返回之后列表还是原来那个列表，可滚动条却是在顶部，也就是说我需要重新向下滑半天屏幕，重新浏览一遍我已经看过并且不感兴趣的商品简介，这时候是不是被恶心得想吐？
So更人性化的方案应该是：返回之后滚动条已经自动滚到我刚才最后浏览到的位置，那么我可以愉快地继续往下找了！

上面这个案例的人性化体现在：

  1. 避免重复的无意义的劳动
  2. 更符合人类直觉
  3. 有效缓解用户的暴怒情绪

说了这么多‘废话’。没错！window.sessionStorage对象就是我们需要的东西！我们只需在单击事件中存储滚动条的位置就可以了！也就是只要计算出单击的标签距离页面顶部的位置即可！

  1. 利用meta标签中信息或者检测移动设备型号确定设备像素比dpr(device pixel ratio)以及缩放比scale，
  2. 根据设备的宽度确定相对大小rem(rem=width/10)，并为htm标签设置font-size：rem px;,
  3. 监听窗口大小变化，如果300ms内大小没有变化，则重新获取设备宽度并为htm设置font-size属性。
  4. 然后在写css的时候只需要把px换算成em就好了，结果值 = 实际px值/rem，比如设备宽度是750px的，那么rem就是75px，如果一个盒子的宽度测得是75px,那么写css的时候写成1em即可。

昨天晚上睡觉前尝试用手绘的办法把公司周围的区域画出来，结果简直不忍直视……..于是今天早上痛定思痛，我识趣地直接用谷歌地图，然后把它导出成PDF，再拿到打印店去打印，然并卵，网页导出的像素太低导致打印出来的也没法看。最后,我爆发了!果断打开了曾经只使用了两天就晾在一边的OneNote,然后直接把图拉进去，然后开始标注。（这里如果有个触屏+触控笔就好了，苦逼的我只能用鼠绘。）

  1. 一共有四个层级，可以新建笔记本，笔记本下面可以新建分区，区里面又可以新建页。如果对应到前端可以这样建：前端（笔记本），HTML、CSS、JavaScript、NodeJS（分区），然后比如可以在CSS分区里建立布局、动画、选择器（页）。是不是很方便？

  5. 还可以共享给别人，或者和别人合作。
  6. 免费！！！

简直就是收集和整理资料的利器，那我为什么用了两天就不用了呢？

缺点：

  1. 当时觉得没有印象笔记好用，因为印象笔记还可以保存微信和微博。

早起日记（21）

发表于 2016-06-19   |   分类于 日记   |     |   阅读次数

我不明白我为什么会想睡觉。前天晚上大概12点睡觉，昨天早上6点钟起床，然后写了日记，查了些资料。在7.30的时候，我就突然困得不行了，于是就上床睡到了8.30，至此我已经睡了7个小时。中午去图书馆的时候，想练习下听力，结果睡着了，从1点睡到了2点多，至此我已经睡了8个小时。晚上吃完饭后去图书馆，没坚持多久就又开始困，然后从6点多睡到了7点多，也就是说我一共睡了9小时。这是晚上睡眠时间不足引起的吗？可是一般我这个年纪不是睡8小时就很足够了吗？

为了找出原因，我昨晚12点睡觉，今早7点起床，然后看下白天会不会困。如果我白天困不困和晚上的睡眠时间没有关系，那我好像必须得去医院看看了。另外会不会是所在环境太安静枯燥导致的呢？最后，我还特意去丁香医生上查了下，发现还真有嗜睡症这种病(。﹏。)
*嗜睡症：对照自己年龄段的睡眠时间，结合以往的睡眠习惯，并且自问以下几个问题，如果有其中一项可能有嗜睡症了。

  1. 有没有在阅读书或报纸时睡着；
  2. 与朋友聊天时是否有入睡情况；
  3. 驾驶时是否有睡着现象，如等红绿灯；
  4. 有没有白天出现不可抑制的想睡的感觉；

看到情况1和情况4，我有被吓到。因为我就是这样的情况，一般我也不会困，可是在特定的时间段看会书就马上开始困了。然后我又查了下，发现：

嗜睡又称过度睡眠，指白天睡眠时间过久或夜晚睡眠时间延长，表现为在安静或单调环境下，经常困乏嗜睡，并可不分场合甚至需要十分清醒的情况下，也出现不同程度、不可抗拒的入睡。
嗜睡症的人白天无法控制地昏睡过去，连续超过三个月；会出现短时间的肌肉丧失张力，而发生猝死倒；亦会短时间进入快速眼动的状态。
一般人需要半小时进入深度睡眠状态，而嗜睡症患者只需1分钟，甚至更短时间。

我倒是没有到昏睡的地步，也没有出现1分钟进入深度睡眠的情况。再观察吧，只能先拿自己做下实验了。另外明天就要去公司实习了，还是有点紧张的。加油吧！

昨天晚上基本上在补预备党员思想汇报，要补4篇，目前已经写了3篇半。在写的过程中我突然意识到如果这是一份汇报，那我到底是在向谁汇报呢？

  • 一开始写的时候，我觉得我自己是在写给一个虚无的党组织看，向它汇报自己的近况，向它宣传党的荣耀，却丝毫不提党的缺陷。
  • 写到第二篇的时候，我突然觉得向它汇报真的太累了，于是我就一半内容写给自己看，一半内容写给它看。
  • 写到第三篇的时候，我就索性沉醉在自己的回忆，向自己对白的场景中了。

为什么会产生这种现象呢？因为一般开头的时候会写一句敬爱的党组织，结尾会写一句请党组织帮助我，指导我....，然而我却感受不到它的真实存在感，让我产生了写这种所谓的思想汇报，压根就是形式所需的感觉。是的，就是形式主义，至少在学校里是这样的。因为写这种思想汇报，除了自己没有人会看，没有所谓的党组织会通过看这个了解你近期的情况，然后帮助你成长，帮你答疑惑。就像你在真诚的写一封信，然而寄出去的时候，收件人却是空白的……..
