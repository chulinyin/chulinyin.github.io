基础知识

web和移动应用：

  1. 用户在浏览器（客户端）里输入或点击一个链接
  2. 浏览器向服务器发送HTTP请求
  3. 服务器处理请求，如果查询字符串或者请求体里含有参数，服务器也会把这些参数信息考虑进去。
  4. 服务器更新、获取或者转换数据库里的数据
  5. 服务器以HTML、JSON、或者其他格式返回一个HTTP响应
  6. 浏览器接收HTTP响应
  7. 浏览器以HTML或者其他格式（如JPEG、XML或者JSON）把HTPP响应呈现给用户。

移动应用用远胜应用取代了浏览器，带宽带来的数据传输限制、更小的屏幕、更高效地使用本地存储。

原始string 和 Sting 对象的区别

每次读取一个基本类型值时，后台会创建一个对应的基本包装类型的对象。

使用 new 操作符创建的引用类型的实例，在执行流离开当前作用域之前都一直保存在内存中。而自动创建的基本包装类型对象，则只存在于一行代码的执行瞬间，然后立即被销毁。所以无法在运行时为基本类型值添加属性或方法。

李笑来老师在「一块听听」上的读书经验分享

发表于 2017-07-25   |   分类于 个人提高   |     |   阅读次数

看了一眼创建日期，居然是去年11月份的了。现在稍微整理一下，发出来吧～

我的读书路径与境界更新

第一个阶段是漫无目的地读书

一开始读书很难做到目标明确，而且一上来就目标明确可能会造成一种后果——失去获得意外好运的机会。能读多少就读多少，能读什么就读什么，不用挑。这样饥不择食读书的好处：

  1. 阅读变成了吃饭一样的需求，不读脑子就饿。
  2. 经常有各种意外的好运，从莫名其妙的地方读到了原本可能不可能获得的知识。
  3. 一不小心就积累了足够的里程 ——你要知道，驾驶经验远比驾驶技巧重要得多。

第二个阶段开始挑书

挑书的能力在大量阅读的过程中会自然地逐步形成。不要向别人索要书单，向别人索要书单是使自己挑书能力不断退化，直至为零的最有效做法。

你的挑书能力不可能不长进，除非你一直让别人帮你挑书。他的经历：

读者文摘、读书、小说、非虚构类书籍、虚构类的快感转向影视、猎奇、符合科学方法论的文字、“有繁殖能力的知识”（有很多知识只是信息）

两个基本价值观：

  1. 需要认清现实并思考未来
  2. 更喜欢有繁殖能力的知识

所以兴趣转向：非虚构类书籍、英文原版书籍、心理学、经济学、策略学、脑科学。

一开始也是抓到就开始读，然后就又会挑了。哪个出版社更靠谱、哪个作者更有趣

第三个阶段读书变少

只读学术文献类书籍，大量阅读学术论文非常锻炼阅读理解能力，因为作者并不关心读者是否看得懂，他只求能够自圆其说，并把事情讲清楚。

第四个阶段检索阅读（有目标地阅读）

电脑上建立图书馆（电子书），知道哪些书里大概有哪些内容，然后用全文检索，找出对自己有用的那部分。

第五个阶段什么都看（不由自主地挑）

了解真实的世界，了解APP，了解人们怎么想。足不出户，却知天下。

读书路径：泛读、精读、泛读；另一个维度的路径：全盘接受、部分接受、不能接受的部分也多想想看。

阅读的目的——改变生活

改变生活，知识改变命运并不是什么空话。比如，学过概率论的人，基本不可能会再去买彩票，因为那是对自己智商的侮辱。所以智商是可以学出来的。比如，女生的经期前后会有情绪起伏是因为激素分泌失调，但是其实可以通过基本没有副作用的药物调节。（找出资料给女生看，消除病耻感，然后去医院配药。）

改变生活的最有效方法就是习得新的技能。

杭州户口、集体户口、农村户口、居住证？？？

发表于 2017-07-23   |   分类于 日记   |     |   阅读次数

实习的时候就在考虑要不要把户口迁移到杭州的事情了，结果家里一边说「要迁」，一边说「不要迁」，而我就正好被夹在中间不知所措，于是便决定自己做一番调研，搞清楚其中的利弊，自己做最终的决定。下面就是我最后整理出来的详细说明，不过我还是想先说说我自己的结论，因为后面的说明确实比较长……

因为考虑到未来10年内杭州很可能发展成现在的北上广，也就是所谓的「新一线」城市，所以只要我们国家的「户口制度」不取消，那么杭州的户口一定会越来越“值钱”（看看在北京和上海落户的难度就知道了）。所以，如果是以下情况之一的，趁现在杭州户口还没缩紧，甚至还有人才引进政策，早点迁吧。（当然，其他城市就不在我的考虑范围之内了，因为各地的发展情况和政策都不太一样，不过这份调研还是可以作为参考的。另外，以下情况也只适用于普通人，对牛逼的人来说，户口什么的不是事，只要负责继续牛逼就好了。）

  1. 如果自己户口在三四五线城市，且不是农村户口，且没有什么福利（比如每年分红、可能会拆迁什么的……）；

  2. 如果考上了杭州的公务员；

  3. 如果两年内在杭州买得起房（搞定首付）；

  4. 如果决定今后在杭州发展；

  5. 如果在一家大公司发展，且收入不错；

    ​

那么，一般人纠结于「迁」还是「不迁」的原因是什么呢？

  1. 可能是原来是农村户口，不舍得农村户口的福利（最基本的包括土地承包、宅基地、农田和山林分配）；
  2. 可能是暂时不确定是否要留在杭州长期发展；
  3. 可能是不知道杭州户口到底有什么好处，或者说不知道没有杭州户口会对自己造成什么影响。

当然，其实更多的可能是，大多数人都没有考虑过「迁户口」这个问题。

PS：如果你关注房价，你就会注意到户口、居住证这种琐事的重要性，因为这会涉及到买房的权限问题。而杭州某些地方的房价，去年年初到今年现在可以说涨了近一倍。也就是说，你早一年和晚一年买的差价，是近百万级别的，百万的概念就是……我也不知道，我又没见过那么多钱，哈哈哈哈。

不过买房这是个蛮现实的问题，目前我国一二线大城市的房子大都还是属于经济学里的「资产项」而不是「负资产」，也就是可以产生现金流的，通俗点来说，想想如果你去年买了一套房，今年卖掉，居然就赚了近百万……

当然，这种事情还是需要自己去判断的，万一跌了呢，虽然目前看着不太可能。

废话不多少，下面就献上我的调研结果吧，也就是各种乱七八糟「户口」的利弊。（真希望我们伟大的祖国母亲可以早日取消这种户口制度，咱们不都是祖国的儿女嘛，在母亲的怀抱里，爱去哪去哪，不受限制多好……）

友情、爱情和亲情之爱情

发表于 2017-07-22   |   分类于 日记   |     |   阅读次数

今天一次偶然的机会，让我重新开始思考去年定下的这个「命题作文」。因此，废话不多说，以下便是我对自己爱情观的阐述。

咔！听说写文章最好能引经据典，因为那样比较有说服力，所以我就先引用几个我比较喜爱的歌手的「爱情观」好了。

崔健在「蓝色骨头」中唱到：「三角架有三条腿才稳定，少了任何一条都要不停的运动。我的生活也要三大要素才幸福，就是为了得到幸运人们才忙活着」，而支撑起幸福的三大要素在崔健看来是「事业」、「身体」和「爱情」。
对于前二者，崔健都明确地知道如何取获得。比如：对于「事业」，崔健认为「能高高兴兴工作挣钱养活自己，有话就说 ，话就写，而且要彻底，因为每次彻底之后才会出现美妙的空虚」；对于「身体」，崔健认为「身体一定要健康，因为身体要是不舒服 什么都是白给，所以我一周三次跑步加上一次游泳，在运动中想事儿是越想越起劲儿」；但是对于「爱情」，崔健却只能说「当你真的爱的时候理论都是虚的，只有分手的时候疼痛才是实的……谁能告诉我爱情到底要我使出多大的力气」。由此可以看出，事业和身体的重要性虽然在爱情之前，但至少它俩都是有法可循的，而爱情却显得扑朔迷离，只能说真的爱的时候就自然知晓了。

林忆莲在「伤痕」中唱到：「为情困，磨折了灵魂。该爱就爱，该恨的就恨，要为自己保留几分」、「有时爱美在无法永恒，爱有多销魂，就有多伤人，你若勇敢爱了，就要勇敢分」。也就是说爱情并不总是那么完美，要敢爱、敢恨、敢分。

罗大佑在「恋曲1980」中唱到：「你曾经对我说，你永远爱着我。爱情这东西我明白，但永远是什么？姑娘你别哭泣，我俩还在一起，今天的欢乐将是明天永恒的回忆」、「现在你说的话都只是你的勇气。春天刮着风，秋天下着雨，春风秋雨多少海誓山盟随风远去」、「你不属于我，我也不拥有你，姑娘世上没有人有占有的权利。或许我们分手，就这么不回头，至少不用编织一些美丽的借口」、「亲爱的莫再说你我永远不分离」。罗大佑在这首歌里的爱情观太过于理性了，但是我很喜欢这种坦白和随性。

妈蛋，终于轮到我出场了！虽然综合前面的“引经据典”，已经基本能够表达我自己的爱情观了。但，我就是我，不一样的烟火，我还是得说说自己对于爱情的想法。咳咳……注意！进入正戏了。

我认为情侣应该是平等的，不应该存在你欠我什么，我欠你什么，或者说我应该给你什么，你应该给我什么。我们只是相互「自愿」地保持这种特殊的关系，我们相互吸引，同时又相互努力着成为对方眼中一个更好的自己。
如果我做了什么让你开心的事，那么那件事肯定也让我开心了，所以在我的爱情观里不存在「我为了你，……」这种说法。因此，我认为当某一方开始不满，觉得自己处于劣势或者付出的一方，并有「我为了你，……，你却……」这种想法的时候，爱情就已经变质了。那么或许也就没有维持这段爱情的必要了。
爱情不是一种交易，而是一种合作。

哈哈，至此我已经写完了我的友情观和爱情观，至于最后的亲情观，我不知道我什么时候才能鼓起勇气写下来，并且公之大众。或许，永远也不会吧……
