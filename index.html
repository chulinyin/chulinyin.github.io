关于责任的一点思考

发表于 2016-12-04   |   分类于 日记   |     |   阅读次数

一点琐事引发的思考：人和人的思考方式天差地别，但是在某些事情上的想法和行为却又出奇的一致。比如有的人倾向于积极地为自己行为所产生的结果负责，而有的人倾向于减轻甚至逃避这种责任。

今天中午，一位室友在晾衣服的时候，衣服了掉下来，不小心打碎了另一位室友的碗。我告诉他，碗碎了。然后他说，“没事，反正他也不用这个碗了。”

这是一件很小的事情，但是这里面所展现出的问题，我却在很多人身上见过。毫无疑问，我的这位室友在这件事情中是属于后一类人的。如果非要较真，他随口说出的那句话无疑是没有道理的，且让人听了不舒服。因为这里面存在这样一种逻辑：“我的行为使你的财产受损，但因为我认为那个财产是你目前不需要的，所以我不必为此负责，或者可以少负一点责任。”这里存在两个问题，第一，你怎么知道这个财产别人不需要了？第二，即使他确实用不到那个财产了，那难道就意味着你可以破坏而不赔偿？

按道理说，如果一个人使他人的财产受到损失，他的第一反应不应该是感到愧疚，表示歉意，然后主动弥补以获得他人的原谅吗？为什么人可以很自然甚至无意识地逃避这种责任呢？而且这似乎是一种普遍现象。

如果行为所产生的结果让自己产生了巨大痛苦，此时内心的自我逃避和免责似乎是可以理解的，因为这样可以防止承受不住压力的精神崩溃。但为什么明明是自己承受范围之内的事情，有些人的意识还是依然会不由自主地逃避呢？

以上想法是在较真情况下的思考。因为当事人没有在场，所以不能说我那室友随口所说的话是他真实的想法。

URL (stable)

方法

url.parse(urlString[, parseQueryString[, slashesDenoteHost]])
  • parseQueryString: 布尔值，默认为 false 。如果为true，则返回的 URL 对象的 query 属性奖总是被设为 querystring 模块的 parse() 方法返回的对象。如果为 false ， 则返回的 URL 对象的 query 属性将是一个未被解析的字符串。
  • slashesDenoteHost: 布尔值，默认为 false 。如果为 true ，则字符 // 之后， 下一个 / 之前的字符串将被解释为 host 。例如，给 //foo/bar ，返回结果将是 {host: 'foo', pathname: '/bar'} 而不是 {pathname: '//foo/bar'} 。

概念

Client) : 客户端，是一块计算机硬件或软件，可以访问由服务器提供的服务。服务器通常在另一个计算机系统上（也可能在本地），在这种情况下，客户端通过网络的方式访问服务。是一个计算机程序，其操作的一部分依靠发送请求到另一个计算机程序（可能在也可能不在另一台计算机上）。比如，web 浏览器就是一个客户端，可用于连接 web 服务器并显示网页。

Server : 服务器，是一个计算机程序或者设备，用于为其他程序或者设备（客户端）提供功能。一个管理资源并为用户提供服务的计算机软件。

web server : 网络服务器是一个通过 HTTP 处理请求的计算机系统。可以指整个系统，或者特指接收并管理 HTTP 请求的软件。

HTTP : HyperText Transfer Protocol ，超文本传输协议。 是在万维网上进行通信时所使用的协议方案。 Web 使用其作为规范，完成客户端到服务器端等一系列运作流程。

Agent : 代理（客户端），发起自动 HTTP 请求的半智能 Web 客户端。通常指客户端程序，如浏览器就是一个 Agent 。

Proxy : 代理，位于客户端和服务器之间的 HTTP 中间实体。接收由客户端发送的请求并转发给服务器，接收服务器返回的响应并转发给客户端。

《思维的乐趣》——知识分子的不幸

发表于 2016-10-15   |   分类于 读书随感   |     |   阅读次数

开始第二遍看《思维的乐趣》了，我会对我觉得有意思的部分做摘录，然后讨论一番，希望能够帮助我在思维中享受到某些乐趣。

什么是知识分子最害怕的事？而且我有答案，自以为经得起全球知识分子的质疑，那就是：“知识分子最怕活在不理智的年代。”所谓不理智的年代，就是伽利略低头认罪，承认地球不转的年代，也是拉瓦锡上断头台的年代；是茨威格服毒自杀的年代，也是老舍跳进太平湖的年代。我认为，知识分子的长处只是会以理服人，假如不讲理，他就没有长处，只有短处，活着没意思，不如死掉。

什么叫理智？——理性智慧？摆事实，讲道理，谈利害？

但知识分子赶上这么个年代，死活不是问题。最大的问题是：这个倒霉的年头几何时过去。假如能赶上这年头过去，就活着；赶不上就犯不着再拖下去。……就我所知，他们最关心的正是赶得上赶不上的问题。在那年头死掉的知识分子，只要不是被杀，准是觉得赶不上好年头了。而活下来的准觉得自己还能赶上——当然，被改造好了、不再是知识分子的人不在此列。

我们所说的不理智，到底是因何而起？对此我有个答案，但不愿为此打赌，主要是怕对方输了赖账：此种不理智，总是起源于价值观或信仰的领域。对于这世界上的各种信仰，我并无偏见，对有坚定信仰的人我还很佩服，但我不得不指出，狂信会导致偏执和不理智。

因为狂信，人就不想讲道理。我个人以为，无理可讲比尸横遍野更糟；而且，只要到了无理可讲的地步，肯定也要尸横遍野，“文化大革命”里就死人不少，还造成了全民知识水平的大倒退。

当然信仰不是总要导致狂信，它也不总是导致不理智。全无信仰的人往往不堪信任，在我们现在的社会里，无信仰无价值的人正给社会制造麻烦，谁也不能视而不见。

我的老师说：对一般人来说，有信仰比无信仰要好。

《思维的乐趣》——中国知识分子与中古遗风

发表于 2016-10-06   |   分类于 读书随感   |     |   阅读次数

开始第二遍看《思维的乐趣》了，我会对我觉得有意思的部分做摘录，然后讨论一番，希望能够帮助我在思维中享受到某些乐趣。

我们国家总以受过某种程度的教育为尺度来界定知识分子，外国人却不是这样想的。他们认为工程师、牙医之类的人，只能算是专业人员，不能算知识分子，知识分子应该是在大学或者研究部门供职，不坐班也不挣大钱的那些人，还要“批判社会”。

维基百科中说：知识分子主要是指具有相对丰富知识，自由思想，独立人格，对于现实持有一定批判精神的人士。在中国大陆知识分子不被视为一种阶级成分。[1]

一般说来，现代关于知识分子有三种定义。第一种定义是指以从事思想、著作以及精神生活等方面职业为主的人士；第二种定义主要来自马克思主义，知识分子是指从事教学、律师、新闻等职业的人员，这种定义在中国大陆比较流行；第三种定义指文化知识分子，主要指具有文化、艺术方面特殊才能的人，因其才能获得某种话语权，从而能够以这种话语权影响公众和其它事物的人士。

维基百科中还有一句比较有意思：在中国，秦汉大一统之后，皇权无可逃避于天地间，国家强过社会，士大夫都是官吏，读书人虽多，知识分子却绝少。

中国知识分子关注社会的伦理道德，经常赤膊上阵，论说是非；而外国的知识分子则是以科学为基点，关注人类的未来，就是讨论道德问题，也是以理性为基础来讨论。人家那里热衷于伦理道德的，主要是些教士，还有一些是家庭妇女。大学教授绝不会站在讲坛上说：你们这些罪人，快忏悔吧……这与身份不符。因为口沫飞溅，对别人大作价值评判，层次很低。

我国的知识分子在讨论社会问题时，常说的一件事就是别人太无知。一般来说，知识的多寡是个客观的标准，但把自编的黑话也列入知识的范畴，就难说有多客观了。（不知道学问，死记些名字，有何必要？）还有一个爱说的话题就是别人“格调低下”，我以为这句话的意思是说：”兄弟我格调甚高，不是俗人！”（挑别人毛病要靠谱，不要借着贬别人来抬自己。）

如果把道德伦理的功能概括为批判和建设两个方面，以上所说的属于批判方面。我不认为这是批判社会——这是批判人。知识分子的批判火力对两类人最为猛烈：一类是在校学生，尤其是中学生；另一类是踩着地雷断了腿的同类。这道理很明白——别人咱也惹不起。

知识分子倾向于批判社会，而我国的“知识分子”倾向于批判人？

我们当然可以潜心于伦理学、道德哲学，营造一批道德体系，供社会挑选，或是向社会推荐——但是这件事几乎没有人干。现在的知识分子都学乖了，只管呼吁不管干，并且善用一种无主句：“要如何如何”。

中国现代知识分子的遗风是什么？他既不像远古的中国知识分子（如孔孟、杨朱、墨子）那样建立道德体系，也不像现代欧美知识分子跨价值观地立论（价值中立）。最爱干的事是拿着已有的道德体系说别人，如前所述，这正是中古的遗风。倒霉的是，在社会转型时期，已有的道德体系不完备，自己都说不清；于是就哀叹：人心不古，世道浇漓，道德武器船不坚，炮不利，造新船新炮又不敢。其实可以把开船打炮的事交给别人干——但咱们又怕失业。当然，知识分子也是社会的一份子，也该有公民热情，针砭时弊也是知识分子该干的事；不过出于公民热情去做事时，是以公民的身份，而非知识分子的身份，和大家完全平等。这个地位咱们又接收不了，非要有点知识分子特色不可。照我看这个特色就是中古特色。

现在中国知识分子在关注社会时，批判找不找目标，颂扬也找不着目标，只一件事找得着目标：呼吁速将大任降给我们，这大任乃是我们维护价值体系的责任，没有它我们就丧失了存在的意义。要论价值体系的形成，从自然地理到生活方式都有一份作用，其功能也是关系到每一个人，维护也好，变革也罢，总不能光知识分子说了算呐。要社会把这份责任全交给你，得有个理由。说来说去只能说凭我清楚明白。那么我只能凭思维能力来负这份责任，说那些说得清的事；把那些说不清的事，交付给公论。现代的欧美知识分子就是这么讨论社会问题：从人类的立场，从科学的立场，从理性的立场，把价值的立场剩给别人。咱们能不能学会？

批判为什么会找不着目标呢？是即使找着目标也要权衡利弊，不敢说出口吧？

颂扬为什么也找不着目标呢？因为真的没有什么值得颂扬的。

可是我们的“知识分子”（那些明明算不上，却自诩如是的人）喜欢先把价值的立场给抢占了？

最后说说中国知识分子的传统。当然，他有“士”的传统。有人说，他先天下之忧而忧，后天下之乐而乐（悲观主义者？），有人说，他以天下为己任（国际主义者？），我看都不典型。最典型的是他自以为道德清高（士有百行），地位崇高（四民之首），有资格教训别人（教化于民）。这就是说，我们是这样看自己的。问题是别人怎样看我们。

这就不光光是知识分子了，许多人都自以为道德清高，有资格教训别人呢！内心的道德优越感，犹如毒品一般，让他们痴狂。

如果我们讨论社会问题，就讲硬道理：有什么事，我知道，别人还不知道；或者有什么复杂的问题，我想通了，别人想不通；也就是说，按现代的标准来表现知识分子的能力。这样虽然缺少了中国特色，但也未见得不好。

不知道这种现象或者说讨论氛围什么时候才能开始在我们国家弥漫。
