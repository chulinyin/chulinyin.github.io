致谢

发表于 2017-05-14   |   分类于 日记   |     |   阅读次数

请了一礼拜假忙学校的事，目前毕业设计实物验已经收官了，论文也写完了，查重后显示地重复率在20%~30%之间（因为不同的数据库，查重结果会有所不同），而学校要求是低于30%，也就是说我抽个时间再改一改就没问题了，也就是说我现在只要准备毕业答辩就好了😄。但我目前纠结的是，PPT是自己做，还是套用模板？

下面是我写在论文致谢部分的几段话，大多是真情独白，当然也有极小部分是讽刺（反话）。

一眨眼，大学四年就这样过去了，这四年的时光充满了各种各样的记忆，有些早已抛之云外，有些却仍然恍如隔日。还记得作为一名大一新生的时候，有许多“老人”感慨时光飞逝、岁月如梭，劝告我们珍惜大学时光。当时的我们可能对此不屑一顾，然而转眼之间我们却已变成那些“老人”。

我作为一名标准的中国大学生，进入大学之前几乎是“两耳不闻窗外事，一心只读圣贤书”的状态。直到进入大学后，才开始逐渐形成自己的价值观、人生观和世界观，发展自己的兴趣爱好，知道自己到底想要干什么。这四年风雨，要感谢的人太多，是在他们的帮助和我自身的努力下，才有了这充实的四年。

感谢班主任高坚老师、辅导员于兹嬴老师和一些学长和学姐在生活、学习等各方面给予的关心和帮助；感谢导师王斌锐老师在做课题、写论文期间给予的帮助，当然，更想感谢的是王老师在做人、做事方面的教诲，王老师是我在大学期间遇到的相当罕见的优秀老师之一；感谢创行社团，让我体会到了一群人为一个共同目标努力的乐趣，让我学习到了“专业、严谨、敢于担当”的做事方式；感谢室友的包容关心以及身边同学和朋友的帮助；最后，感谢罗永浩老师对我三观上的引导，帮助我度过了最困难的迷茫期。

从人的一生来看，四年不过是弹指一瞬间，但这黄金四年对未来人生的影响却是至关重要，再次感谢所有帮助我成长的人。
