杭州户口、集体户口、农村户口、居住证？？？

发表于 2017-07-23   |   分类于 日记   |     |   阅读次数

实习的时候就在考虑要不要把户口迁移到杭州的事情了，结果家里一边说「要迁」，一边说「不要迁」，而我就正好被夹在中间不知所措，于是便决定自己做一番调研，搞清楚其中的利弊，自己做最终的决定。下面就是我最后整理出来的详细说明，不过我还是想先说说我自己的结论，因为后面的说明确实比较长……

因为考虑到未来10年内杭州很可能发展成现在的北上广，也就是所谓的「新一线」城市，所以只要我们国家的「户口制度」不取消，那么杭州的户口一定会越来越“值钱”（看看在北京和上海落户的难度就知道了）。所以，如果是以下情况之一的，趁现在杭州户口还没缩紧，甚至还有人才引进政策，早点迁吧。（当然，其他城市就不在我的考虑范围之内了，因为各地的发展情况和政策都不太一样，不过这份调研还是可以作为参考的。另外，以下情况也只适用于普通人，对牛逼的人来说，户口什么的不是事，只要负责继续牛逼就好了。）

  1. 如果自己户口在三四五线城市，且不是农村户口，且没有什么福利（比如每年分红、可能会拆迁什么的……）；

  2. 如果考上了杭州的公务员；

  3. 如果两年内在杭州买得起房（搞定首付）；

  4. 如果决定今后在杭州发展；

  5. 如果在一家大公司发展，且收入不错；

    ​

那么，一般人纠结于「迁」还是「不迁」的原因是什么呢？

  1. 可能是原来是农村户口，不舍得农村户口的福利（最基本的包括土地承包、宅基地、农田和山林分配）；
  2. 可能是暂时不确定是否要留在杭州长期发展；
  3. 可能是不知道杭州户口到底有什么好处，或者说不知道没有杭州户口会对自己造成什么影响。

当然，其实更多的可能是，大多数人都没有考虑过「迁户口」这个问题。

PS：如果你关注房价，你就会注意到户口、居住证这种琐事的重要性，因为这会涉及到买房的权限问题。而杭州某些地方的房价，去年年初到今年现在可以说涨了近一倍。也就是说，你早一年和晚一年买的差价，是近百万级别的，百万的概念就是……我也不知道，我又没见过那么多钱，哈哈哈哈。

不过买房这是个蛮现实的问题，目前我国一二线大城市的房子大都还是属于经济学里的「资产项」而不是「负资产」，也就是可以产生现金流的，通俗点来说，想想如果你去年买了一套房，今年卖掉，居然就赚了近百万……

当然，这种事情还是需要自己去判断的，万一跌了呢，虽然目前看着不太可能。

废话不多少，下面就献上我的调研结果吧，也就是各种乱七八糟「户口」的利弊。（真希望我们伟大的祖国母亲可以早日取消这种户口制度，咱们不都是祖国的儿女嘛，在母亲的怀抱里，爱去哪去哪，不受限制多好……）

注：以下个别政策可能会变，所以真正要办理XX的时候，最好还是去看政府发布的最新文件……不过用于了解一下「集体户」、「农村户口」、「居住证」……的概念还是足够了。

杭州集体户口

因为就算迁移户口到杭州，目前大多数人也只能迁移成为「杭州集体户口」，因此先来说说「杭州集体户口」和「真正杭州户口」的区别。

杭州集体户口和杭州户口在福利和权力方面没有什么区别，只不过杭州集体户口不是永久的，会存在以上的图片中的限制。

利：

  1. 办证件和护照方便；

  2. 摇车号更方便；

  3. 经济适用房、廉租房、公租房申请；

  4. 买房可以享受政策优惠，如贷款比例更高、不需要满足一定的社保缴纳年数、不限购等；

  5. 孩子以后的生活（PS：上学什么的）；

  6. 以后可以按照杭州的养老保险标准领取养老保险；

  7. 杭州户口的新生儿，出生就可以办少儿医保；

  8. 一人入户，全家即可全入户。(3年后到公安局办理随迁)；

  9. 杭州户口居民可以更方便地贷款买车、买房和做生意；

  10. 可申请保障性住房（结婚后可以申请杭州的经济适用房）；

  11. 社保采用杭州标准（以后可以按照杭州的养老保险标准领取养老保险）

  12. 买房：不限购！可以买两套。之前限购令，外地户口需要连续交满一年的社保才能买房（PS：今年好像又变成2年了），而杭州户口就可以直接买，现在限购令虽然已经部分取消，但是买140方以下的房子仍然是一样的政策；

    买车：限牌阶段，需要杭州户口或者社保连续交满两年才可以参加车牌摇号（PS：现在居住证也能摇号的）；

  13. 杭州对本地户口的就业、创业人员支持政策很大，相关税收优惠、办公场地租金补贴都有很多！

总之就是杭州户口能够享受到的政策福利，杭州的集体户口也大都能享受到。

弊：

  1. 如果是从农业户口变成非农户口，意味着农村的大学毕业生将失去承包土地的权利， 失去继承父母所承包的土地使用权、 宅基地， 以及自留地，而且农转非后办的集体户口，不能回原籍；

  2. 如员工需要户口的相关证明，或辞职，要办理很多手续，如结婚、生育、办理身份证换代等（PS：这条我也不太清楚什么意思）。

    ​

农村户口

利：

  1. 分田、分地、宅基地、自留地；
  2. 农村户口到城市工作，和城市人口享有同样的劳动保险待遇（PS：就是几险几金吧），可以迁移，到岁数和城市户口人一样退休，拿一样退休金。并且，农村有田，可以自己种，可以包给别人种，也可以什么都不种，就那样闲着，因为没有皇粮（搁以前，你有一亩地，一年必须缴多少粮或钱，现在没有了，而且种粮的话还有补助）。户口在村里，年底可能还有分红。自己的宅基地盖房，没有房产税，没有70年到期一说，没有土地出让金。

弊：

  1. 办理一些手续需要回户口所在地办理；
  2. 应届本科生过了12月份，杭州变成积分落户，落户门槛越来越高。
  3. 以后买房会受到限制，如贷款比例、社保缴纳年数、限购等；
  4. 以后想要买房落户，必须满足很多条件（购房金额要求：上城区、钱江新城购房单套总价100万元以上(含100万元);经济开发区(下沙)购房单套总价60万元以上(含60万元);其它地区购房单套总价80万元以上(含80万元),同时需付清房款）；
  5. 限购：自购房之日起前3年内在本市连续缴纳社保2年以上，并且不得通过补缴个人所得税或社会保险购买住房。

如果你舍不得农村户口，那么办个「居住证」还是很不错的。（不过目前杭州的没有北京那样的工作居住证，北京的工作居住证可以享受和北京户口差不多的待遇，买房买车都没什么问题，就是孩子在北京上学会受限制，比如高考要回原籍考。而目前杭州的居住证摇号买车没有问题，但买房好像是有限制的。）

杭州居住证

对！还有「居住证」这种东西。「临时居住证」、「居住证」、「人才居住证」、「工作居住证」，我也有点傻傻分不清楚。因为我最后决定先办理个「人才居住证」，因此对于这个居住证的资料会更加详细一点。

利：

根据《通知》规定，持《浙江省居住证》的流动人口在杭依法享受以下权利：

  1. 劳动就业；
  2. 参加社会保险；
  3. 缴存、提取和使用住房公积金；
  4. 依法加入工会和参与社会事务的权利。

并按照国家和省有关法律法规规定，在居住地享受下列基本公共服务：

  1. 义务教育；
  2. 基本公共就业服务；
  3. 基本公共卫生服务和计划生育服务；
  4. 公共文化体育服务；
  5. 法律援助和其他法律服务；
  6. 国家和省人民政府规定的其他基本公共服务。

《浙江省居住证》持有人在居住地按照规定享受下列便利：

  1. 换领、补领居民身份证（PS：可以异地办理证件还是蛮有用的，毕竟政府办证部门的工作效率你懂的，跑回原籍办的话得整死人）；
  2. 办理出入境证件；
  3. 机动车登记；
  4. 申领机动车驾驶证；
  5. 报名参加职业资格考试、申请授予职业资格；
  6. 办理生育服务登记和其他计划生育证明材料；
  7. 国家和省人民政府规定的其他便利。

《人才居住证》持有人员，可享受以下相关待遇：

持有人才居住证可享受以下相关待遇
(一) 参加本市专业技术职务任职资格评审或考试、执业(职业)资格的考试或登记。
(二) 在本市参加国家职业培训考核鉴定。
(三) 安排子女在本市就读。幼儿教育及义务教育阶段，由居住地所在区教育行政部门安排到相应条件的学校就读;高中阶段，参加全市统一的招生考试，由市教育行政部门安排到相应条件的学校就读。
(四) 在规定范围内报考本市公务员。
(五) 按规定参加本市基本养老保险。持有《人才居住证》的人员，在杭州市区就业后，参加本市机关事业单位养老金统筹或企业职工基本养老保险。今后，需转移养老保险关系的，按有关规定办理。
(六) 按规定参加本市基本养老保险的人员，可以参加本市城镇职工基本医疗保险。离开本市时，本市的医疗保险经办机构应当将其基本医疗保险关系和个人医疗帐户储存额转移到其户籍所在地的基本医疗保险经办机构;当地未建立社会保险机构的，将其个人医疗帐户储存额及其利息一次性支付给本人。
(七) 按规定在本市缴存和使用住房公积金。已在户籍所在地缴存住房公积金的，可以将在户籍所在地缴存的住房公积金余额转入本市住房公积金帐户，并按本市的有关政策规定缴存和使用住房公积金。离开本市时，可按规定办理职工住房公积金帐户存储余额转移手续。
(八) 在本市工作并居住1年以上的，可以按照公安部的有关规定，办理因私出国商务手续。
(九) 申请办理驾驶证或临时驾驶证以及机动车注册登记手续；换领、补领居民身份证；办理出入境证件；
备注 国家、省和本市对持有《人才居住证》人员的待遇另有规定的，从其规定。

弊：

  1. 养老金和医保，退休或者返回原籍后单位缴纳部分（22%、12%）会计入社会统筹，只有个人承担（8%、2%）的会转回；
  2. 一年有效期，而户口是没有有效期这种说法的；
  3. 工作单位变更和租房变更都需要去更新居住证；
  4. 买房目前还是有限制的。

办理条件

  1. 非杭州市区户籍人员拟在杭州市区居住三日以上的；

  2. 浙江省流动人口在杭州居住地居住半年以下的；

  3. 符合办理人才居住证的（具有普通高校本科及以上学历的人员）

    ​

办理材料（原件和复印件，原件即验即退）

  1. 填写《杭州市人才居住证申请表》)一式三份；

  2. 有效的身份证明（身份证或护照）；

  3. 近期白底免冠彩色1寸证件照片3张（其中1张用于制作《人才居住证》）；

  4. 与用人单位签订的1年以上期限的劳动（聘用）合同；

  5. 企业营业执照复印件盖公章或事业单位法人证书；（PS：7月开始不需要了）

  6. 本人的学历学位证书、专业技术等级证书、国家职业资格证书或者业绩证明材料；

  7. 在本市市区的住所证明（购房产权证、租房合同及业主房产证复印件、单位出示住所证明及住所房产证复印件）

  8. 本市二级以上医院出具的近期健康状况证明（体检表或健康证）；（PS：7月开始不需要了）

    ​

办理流程

1、申请办理

2、提供申办材料

3、居住地派出所或者社区服务机构签发、制证（杭州市各区派出所联系方式）

4、领证

15个工作日后，去办理地点领取。

注意：每年签注一次。逾期未办理签注，使用功能中止。

办理时间及费用

  1. 办理时间：15个工作日
  1. 有效期：注意每年签注一次。逾期未办理签注，使用功能中止
  2. 费用：首次申领免工本费

####

问答

问：杭州人才居住证有什么优势？
答：《浙江省引进人才居住证》就是《杭州市人才居住证》，人才居住证相当于你在杭州的绿卡，其待遇和本地人基本上没有区别，而一般的临时居住证（暂住证）相当于普通签证，有效期一般只有一年，最长三年。人才居住证可以用来摇号。如果社保不到2年，或者中间有中断，可以用这个证来顶替。可以直接参加摇号。具有《杭州市人才居住证》的来杭创业和引进人才子女，享受与本市居民子女同等待遇。在学前教育阶段和义务教育阶段与本市居民子女同等享受相关政策；
