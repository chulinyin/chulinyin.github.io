关于《霸王别姬》

发表于 2017-03-05   |   分类于 观影随感   |     |   阅读次数

是的，我又要开始更新我的博客了，这段时间丢下的东西会慢慢开始拿起来。起初丢下是因为认为自己有更重要的事情需要去完成，没有多余的时间来更新博客。然而事实是我高看自己了，其实自己并没有想象中那么忙碌，而且那些所谓更重要的事情，到现在也没有得到自己想要的结果。因此决定还是脚踏实地，慢慢完善提高自己吧~

前几天看了由陈凯歌执导，张国荣、张丰毅、巩俐主演的电影 ——《霸王别姬》，其中的好几个场景都引人深思。我说几个让我印象深刻的吧~

一、小豆子总是把戏文“我本是女娇娥，又不是男儿郎”记成“我本是男儿郎，又不是女娇娥”，并因此挨了不少打。我想这不是因为小豆子记性不好导致的，而是他自己内心深处的抗拒。按照一般人的思维，那不就是一句台词嘛，又不能当真，何必因此受苦呢？但显然小豆子并不是这样想的，他知道自己会入戏。正如后面段小楼（小石头）对他的评价——“不疯魔不成活”。

二、小赖子带着小豆子逃跑，师兄小石头追了上去，小豆子却突然转身停了下来，告诉小石头席子底下还有三个大子，让他别忘了。显然小石头被惊到了，思考了一会后，流着泪把衣服砸向小豆子，说：“反正你废了，滚吧！”。还有后面小石头流着泪用烟斗搅小豆子的嘴，还说：“谁叫你回来啦？我叫你错！我叫你错！还回来干什么？”。

在这两处场景中，小石头都流下了眼泪。第一处应该是因为感动和不舍；第二处则是因为心疼和不忍。但“奇怪”的是，小石头在这两次心理斗争中都说出了与自己真实内心情感相背的话，做出了相背的举动。

三、小赖子和小豆子出逃后去看了一场名角的戏，看到中途，小赖子嚎啕大哭，说出了一句让我印象十分深刻的话——“他们怎么成的角儿啊？得挨多少打呀？我什么时候才能成角儿啊？”，此处解释了后面小赖子为什么选择了自杀。他在回去的路上说道：“吃了糖葫芦，我就是他妈角了。我他妈怕谁啊？”，可当他看到小豆子被狠狠挨打的场景后，想到自己成角前需要承受的苦，他吞下了所有的冰糖葫芦，然后选择了上吊自杀。小豆子在看戏的时候同样默默流泪，我不知道他想的是否和小赖子一样，但从他后面做出的选择来看（也就是主动认罪，替师兄挨打），他的内心所想应该还是和小赖子不同的。

四、在1937年“七七事变”前夕，此时小豆子和小石头都已经成角，并分别改名为程蝶衣和段小楼。当时街上到处都是学生的抗议游行，喊着“打倒日本帝国主义”的口号。程蝶衣和段小楼拍完照片从楼上走下来的时候，学生围住了他们，认出他们是戏子后，居然说“眼瞅就要当亡国奴了，你们知道吗？这妖里妖气的，你们唱什么戏？没国没家的，你们有没有中国人的良心？”，而这时候程蝶衣和段小楼的经纪人那爷说：“中国人不打中国人！都是一个老祖宗！老祖宗错的了吗？错不了！错不了！”，而群众居然也跟着喊。这幕场景让我很是震惊，群众的脑子呢？

而后面段小楼的话也非常有意思“一个个都他妈忠臣良将的模样，这日本兵就在城外头，打去呀！感情欺负的还是中国人！”，然后旁边的经纪人那爷说：“瞎哄呗，学生们不都没娶过媳妇吗？火气壮，又没钱找姑娘，总得找个地界煞煞火不是？”，这个场景其实现在每天都在网络上重演，只是换了一群人，换了一种场合，换了一种交流方式，但本质还是一样的。

五、共产党掌权后，段小楼和程蝶衣的亲徒弟小四想要改变戏剧形式，认为现代戏也是京戏，而程蝶衣听后先是很耐心地解释说：“现代戏有意思，可服装有点怪，没有京戏的行头好看，布景也太实了……京戏得好看、美……”，但徒弟小四顶嘴说：“我不明白，为什么古时候的英雄美人上了台，就是京戏。现在劳动人民上了台，就不是京戏了？”，程蝶衣听后很是愤怒地说：“你说的是两码事！放肆！”。

再之后程蝶衣罚小四跪下，用的是他师傅的方式。小四这时候却说：“罚我跪，你犯法！”，程蝶衣怒而说：“不罚？不罚你永远是下三滥！还想成角？做梦！”，小四听后摔盆而起说：“对，你领我来，哪儿是想让我成角儿？你就是想找个小力笨儿，小催帮儿小跟包，小腿子，小龙套！”，程蝶衣听后用刀背怒打小四，之后还想让他跪下。小四却说：“师傅，永没那日子啦！”。之后小四就开始迫害他的两个师傅，让他们受尽了折磨。

这里我感到很奇怪，难道程蝶衣认为师傅当初这样对他是对的？他认同”要想人前显贵，就得人后受罪“中的受罪就是犯错就用体罚吗？为什么不受身体上的虐待就成不了角了？还有这个场景中，新老两代人之间发生了剧烈的价值观冲突，其中的原因引人深思。

六、文革时期，那一段人性扭曲的时光。

七、电影最后，当小楼重提那句台词后，程蝶衣好像突然惊醒了，意识到自己是男儿郎，这段感情注定没有结果。最后微微一笑，以虞姬的身份拔剑自刎。

八、我有一点想不通，从电影开始时蝶衣和小楼的对话可以看出小楼的少记了一年，为什么呢？

对，忘记提了，菊仙这个人物也很有意思。
